Pour que l'application fonctionne, je recommande de créer un environnement de travail et d'y installer chaque package de requirements.txt.  

Ensuite, remplissez les fichiers csv et exécutez le script import_data.py.

Ouvrez 3 terminaux et connectez vous à MongoDB sur les ports 27017, 27018 et 27019.

Lancez le script migrate_flat.py

Ouvrez un quatrième terminal et effectuez cette commande : python manage.py runserver
